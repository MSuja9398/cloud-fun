Getting started
3 steps to get running
Follow these to connect your Genesys Cloud bot to this portal.
Step 01

Get your Embed Script

In Genesys Cloud, go to Admin → Messenger → Deployments. Copy the JavaScript snippet from your deployment configuration.

Step 02

Paste into this file

Find the <!-- GENESYS EMBED --> comment below and replace it with your snippet from the Genesys dashboard.

Step 03

Host on GitHub Pages

Push this index.html file to a GitHub repo and enable Pages under Settings → Pages. Your portal is live instantly.
